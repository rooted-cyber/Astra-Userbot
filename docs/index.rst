Astra Userbot Documentation
===========================

Astra Userbot is a Python-based WhatsApp userbot framework designed to simplify
automation, administration and fun interactions on your personal account. It
ships with a plugin system, command parsing, and helpers for media, translation,
and more.

Version: 0.0.1b8

Features
--------

* Modular plugin architecture
* Command decorator and parsing system
* Built-in utilities (wiki, translate, media download, spam, etc.)
* Async operation via WhatsApp client library
* Configuration with environment variables
